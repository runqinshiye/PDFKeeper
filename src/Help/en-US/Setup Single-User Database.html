Setup Single-User Database

SQLite is the database engine used by PDFKeeper for single-user document storage. It provides an easy to setup database option that does not require a database management system.

The single-user database is contained in a file named PDFKeeper.sqlite that is created in %APPDATA%\Robert F. Frasca\PDFKeeper. This database will need to be backed up regularly in the event the database needs to be restored due to corruption or accidental data loss. You are fully responsible for this database!

Setup Single-User Database Now!

The default location for the database may not be ideal for some system configurations and can be moved to an alternate location.

To move the database, perform the following steps from the main form:

  1. Select the Tools menu, and then select Move Database; or press Alt + T, and then M on the keyboard.

  2. Select the folder location where to move the database.

  3. Select OK or press Enter on the keyboard to perform the move.
