Set Category on Documents

To set the category on one or more retrieved documents, perform the following steps:

  1. In the Documents list, click on the check box for each document that you wish to set the category on. To select all documents, select the File menu, then Select, and then All; or press Alt + F, then L, and then A on the keyboard.

  2. Select the File menu, and then select Set Category; or press Ctrl + Shift + C on the keyboard.

  3. On the Set Category dialog, select the desired category or leave blank to clear, and then select OK or press Enter on the keyboard to perform the update.

Additional Information:

  • You can navigate the Documents list using the up and down arrows on the keyboard and check/uncheck a document using the space bar on the keyboard.

  • To uncheck all documents, select the File menu, then Select, and then None; or press Alt + F, then L, and then N on the keyboard.
