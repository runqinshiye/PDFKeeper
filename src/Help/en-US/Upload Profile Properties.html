Upload Profile Properties

The Profile section of the Upload Profiles dialog is where the properties are defined for the upload profile being added or edited.

  1. Enter or edit the Name of the profile. Skip if you wish to set the Name to the combined values of Author and Subject.

  2. Enter or edit the Title, or select a token.

  3. Enter, edit, or select the Author. The Author combo box is linked to the database to allow for selecting an existing Author.

  4. Enter, edit, or select the Subject. The Subject combo box is linked to the database to allow for selecting an existing Subject.

  5. To set the Name of the profile to the combined values of Author and Subject, select Set Profile Name to Author and Subject.

  6. Enter or edit the Keywords (optional).

  7. Enter, edit, or select the Category (optional). The Category combo box is linked to the database to allow for selecting an existing Category.

  8. Select the Tax Year (optional).

  9. Check or uncheck Flag Document. As an example, checking this option is useful when the document type is a bill, making it easy to list bills that require payment.

  10. Check or uncheck OCR each PDF page containing text and image data. When checked, each PDF page that contains one or more images will be processed by OCR to extract text. This is useful when the PDF contains pages with embedded images that contain text that would normally not be processed.

  11. To save the profile, select Save or press Alt + S on the keyboard. To discard the changes, select Discard or press Alt + C on the keyboard.
