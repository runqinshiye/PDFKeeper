Adding, Editing, and Deleting Upload Profiles

The Upload Profiles dialog is where Upload Profiles are added, edited, and deleted.

To add a new profile:

  • Select New or press Alt + N on the keyboard. After doing so, the Profile section of the dialog will enable.

To edit an existing profile:

  • From the drop down list, select the profile that you wish to edit, and then select Edit or press Alt + E on the keyboard. After doing so, the Profile section of the dialog will enable.

To delete an existing profile:

  • From the drop down list, select the profile that you wish to delete, and then select Delete or press Alt + D on the keyboard.

Additional Information:

  • Upload Profile XML files are stored in %APPDATA%\Robert F. Frasca\PDFKeeper\UploadProfiles.

  • Upload Profile folders are stored in %APPDATA%\Robert F. Frasca\PDFKeeper\Upload.

  • Upload Profile XML file will be deleted to the Windows Recycle Bin when performing a delete operation. It is NOT recommended to configure the Windows Recycle Bin to remove files immediately when deleted.
