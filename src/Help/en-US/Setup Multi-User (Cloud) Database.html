Setup Multi-User (Cloud) Database

PDFKeeper can be setup to use a compatible, multi-user (cloud) database for document storage.

At this time, the only multi-user (cloud) database that PDFKeeper has been designed for is Oracle Autonomous Database. Future releases may include support for additional databases based on user request and the feasibility of implementation. For a database to be considered, please open a new issue on the Issue Tracker.

As part of their cloud offering, Oracle offers a Free Tier which is perfect for those who wish to try out PDFKeeper with a cloud database – for free!
