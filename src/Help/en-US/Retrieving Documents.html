Retrieving Documents

From the main form, documents can be retrieved by Search Term, Selections (Author/Subject/Category/Tax Year), or Date Added. In addition, flagged documents or all documents can be listed.

How do you want to retrieve documents?

By Search Term:

  1. Select Find Documents by Search Term.

  2. Enter the search term or select a previously entered search term. For examples, see Find Documents by Search Term.

  3. Select the Find button or press Tab on the keyboard. After doing so, any documents retrieved will be displayed in the Documents list.

By Selections (Author/Subject/Category/Tax Year):

  1. Select Find Documents by Selections.

  2. Select either an Author, Subject, Category, or Tax Year.

  3. If you wish to refine your selection(s), select one or more of the remaining drop down boxes which will only contain items that match the item selected in any previously selected drop down box.

  4. Select Find by Selections or press Alt + Y on the keyboard. After doing so, any documents retrieved will be displayed in the Documents list. To clear search selections and start over, select Clear Selections or press Alt + C on the keyboard.

By Date Added:

  1. Select Find Documents by Date Added.

  2. From the drop down, select the Date. After doing so, any documents retrieved will be displayed in the Documents list.

Flagged Documents:

  • Select List Flagged Documents. After doing so, any documents retrieved will be displayed in the Documents list.

All Documents:

  • Select List All Documents. After doing so, any documents retrieved will be displayed in the Documents list.

Additional Information:

  • When the database contains one or more documents that are flagged, PDFKeeper will display a red flag on the right hand side of the status bar.

  • If you wish to show Flagged Documents on startup of PDFKeeper, please see Setting PDFKeeper Options.

  • In a multi user environment, with a database that contains thousands of documents, it may be necessary to limit who can list all documents. In some environments, listing an entire database table can have an adverse effect on database performance. For instructions, please see, Remove List All Documents Retrieval Action by Policy.
